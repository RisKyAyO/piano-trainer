# Piano Trainer

Un outil web pour **visualiser les gammes et accords sur un clavier de piano**. Selectionne une tonique et un pattern, les notes s'allument automatiquement.

> Outil personnel cree pour apprendre la theorie musicale en pratiquant le piano.

## Fonctionnalites

- 7 gammes : Majeure, Mineure naturelle & harmonique, Pentatonique M/m, Blues, Dorien
- 7 types d'accords : Maj, min, dim, aug, 7e Dom., maj7, min7
- Clavier visuel 2 octaves avec touches noires
- Mode aleatoire pour s'entrainer
- Formule des intervalles affichee (T = ton, d = demi-ton)
- Tonique en bleu fonce, autres notes en bleu clair

## Utilisation

Ouvre simplement `index.html` dans ton navigateur -- aucune dependance, aucune installation.
